## 1. 概述

本文档基于 PRD v0.7-draft 定义 MVP 版本的测试用例，覆盖功能测试、用户验收测试、API 测试和集成测试场景。

### 1.1 测试范围

MVP 版本覆盖的用户故事：
- US-D01a: 基于模板创建任务
- US-D01b: 任务调试
- US-D02: 任务执行监控
- US-D03: 人工干预任务
- US-D06: Provider 选择与配置
- US-A01: 租户管理
- US-A02: 任务模板管理
- US-A03: 能力注册管理
- US-A04: Provider 管理（管理端）
- US-O01: 任务监控
- US-O02: 异常处理

### 1.2 测试优先级定义

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，必须通过 |
| P1 | 重要功能，应该通过 |
| P2 | 一般功能，可选通过 |
